# Perplexity Thread Exporter - Chrome拡張機能

Perplexity AIの過去のスレッドをMarkdown形式でエクスポートするChrome拡張機能です。

## インストール方法

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`chrome_extension`）を選択

## 使い方

### 事前準備
1. ChromeでPerplexity（https://www.perplexity.ai/）にアクセス
2. Googleアカウントでログイン

### 現在のスレッドを保存
1. Perplexityで保存したいスレッドを開く
2. 拡張機能アイコンをクリック
3. 「📄 現在のスレッドを保存」をクリック
4. Markdownファイルがダウンロードされます

### 全スレッドを保存
1. Perplexityのいずれかのページを開く
2. 拡張機能アイコンをクリック
3. 「📚 全スレッドを保存」をクリック
4. 自動でライブラリを巡回し、全スレッドをダウンロードします

## 保存場所

ダウンロードフォルダ内の `perplexity_export/` に保存されます：
- `_index.md` - スレッド一覧
- `0001_タイトル.md`, `0002_タイトル.md`, ... - 各スレッド

## 注意事項

- Perplexityにログインした状態で使用してください
- 全スレッドのエクスポートは時間がかかる場合があります
- エクスポート中はタブを閉じないでください
